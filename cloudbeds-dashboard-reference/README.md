## Design patterns to retain

- One dominant Activity surface, supported by a compact date card and occupancy card.
- Cool neutral canvas, white surfaces, deep neutral text, one blue action/selection color, and restrained teal occupancy/completion accents.
- Activity totals behave as tabs; only the selected tab receives a colored outline and soft fill.
- Reservation records are spacious table rows separated by hairlines, not nested cards.
- One clear primary workflow button per row; secondary actions use quiet line icons.
- Alerts appear as tiny dots or subtle labels instead of a collection of colored badges.
- Forecast and detailed operational views remain separate from the primary daily dashboard.
